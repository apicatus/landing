INTRODUCING API DIGESTORS

Apicatus is a tool designed to assist you as you develop your api. Provides various functions that facilitate each of the processes that make the development of an API, maturing along with it. It lets you create and simulate resources and entities, It also monitor all of it's key performance indicators, allowing you to spot potential problems and bugs.
